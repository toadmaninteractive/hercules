2.2.0

Hercules is now a self-contained .NET Core application and doesn't depend on the .NET Framework version installed in the system.
32bit systems are no longer supported.

Document Editor

  • Previously, when the field value was missing or invalid in the document, Hercules would silently overwrite it with the default value. A user could just not be aware of the fact that his document is actually invalid. Now Hercules always marks values as modified in cases when keeping an existing or missing value would make a document invalid for client applications.
    See Manual for details.
  • Such documents with missing or invalid values are now reported by Find Invalid Documents command.
  • If you edit an element value and then edit it back manually (not with Undo), it is no longer marked as modified.
  • It is now possible to undo past document save. (But undo history is still cleared on schema updates.)
  • Revert command in form element context menu.
  • Tab stop behaviour improved.
  • Support for editing Unreal Engine blueprint class paths, see Manual for details.

Inherited (Linked) Documents (Beta)

  • Design of document inheritance has changed. Instead of a JSON patch, an inherited document now stores both its own JSON and base document's JSON. When base document changes, the inherited document should be rebased over the newer version. This approach allows existing clients to use or modify Hercules data without need to support JSON patches. For example, any game can start using document inheritance without any source code modifications.
    See Manual for details.
  • Configurable behaviour for JSON rebasing, in particular atomic values and list item keys.
  • Inherited values are now displayed semi-transparently. Previously text color was used for this purpose, which made it impossible to find out the status of modified value (it would always be green).
  • Link to another document document toolbar button allows to link an existing document to another document.
  • Revert to base document command in form element context menu for inherited documents.
  • When inherited or based documents change category, documents are unlinked.

Diagrams

  • Property Editor: support dictionaries.
  • Multiple bug fixes of Undo/Redo behaviour.
  • Multiple bug fixes of form-diagram synchronization.
  • Support for non-variant records as diagram blocks.

Curve Editor

  • Scale tool.
  • Allow to setup axis labels.
  • Escape now closes popup windows.
  • UI redesign.

Other Improvements

  • Optimize application startup time.
  • New Development application update channel for QA. Don't use it in production.
  • Split manual to two chapters: Hercules for Designers and Hercules for Developers.
  • Help/Online Help menu option.
  • View Release Notes link in application update dialog.
  • F2 hotkey for Rename Document.
  • Ctrl+E hotkey for Find in Database Explorer.
  • Update JavaScript engine to partially support ECMAScript 6.0.

  • Fix search in deeply nested variants.
